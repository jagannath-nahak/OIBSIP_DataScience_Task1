📊 EDA on Retail Sales Dataset

📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Retail Sales Dataset to uncover meaningful insights about customer behavior, product performance, and sales trends using Python.

The goal is to analyze the dataset and extract business-driven insights through visualizations.

🎯 Objectives

✔ Understand sales distribution
✔ Analyze customer demographics
✔ Identify top-performing categories
✔ Detect purchasing patterns
✔ Visualize business insights

🗂 Dataset Features

The dataset includes:

Customer Age
Gender
Product Category
Quantity Purchased
Price Per Unit
Total Amount
Purchase Date
🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn

📌 Customers aged 25–40 contribute most sales
📌 Female customers show higher purchasing trends
📌 Quantity purchased impacts total revenue
📌 Sales show seasonal variations
